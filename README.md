# BizDev Automatater #


### What is this repository for? ###

The point of this project is to automate some of the weekly biz dev tasks, in particular the transformations of the data from hubspot.

### How do I get set up? ###

* Summary of set up
* Configuration
* Dependencies
* Database configuration
* How to run tests
* Deployment instructions
This program assumes that there will always be a sheet named: 'Pipe per stage'
This also assumes that there will be columns named: 'Associated Company', 'Deal Name', 'Deal Stage', 'Close Date', 'Deal owner', ' Amount ', and 'Deal Description'
These names must be typed exactly as shown
